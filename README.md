# Bioinformatics Projects

**M.Sc. Medical Bioinformatics** | Sri Ramachandra Institute of Higher Education and Research, Chennai

A collection of bioinformatics projects demonstrating computational biology skills, statistical analysis, and programming.

---

## Semester 1 Projects

| # | Project | Language | Domain | Description |
|---|---------|----------|--------|-------------|
| 1 | [Gene Expression Analysis](semester-1/01_gene_expression_analysis) | R | Cell Biology | Statistical analysis of gene expression data, t-tests, volcano plots |
| 2 | [Enzyme Kinetics Analysis](semester-1/02_enzyme_kinetics_analysis) | R | Biochemistry | Michaelis-Menten kinetics, Km/Vmax calculation, curve fitting |
| 3 | [DNA Sequence Analyzer](semester-1/03_dna_sequence_analyzer) | C | Molecular Biology | FASTA parser, nucleotide counting, GC content analysis |
| 4 | [Protein Composition Analyzer](semester-1/04_protein_amino_acid_analyzer) | C | Biochemistry | Amino acid frequency, molecular weight, hydrophobicity analysis |

## How to Run

### R Projects
```bash
cd semester-1/01_gene_expression_analysis/src
Rscript analysis.R
```

### C Projects
```bash
cd semester-1/03_dna_sequence_analyzer/src
gcc -o dna_analyzer dna_analyzer.c
./dna_analyzer ../data/sample1.fasta
```
